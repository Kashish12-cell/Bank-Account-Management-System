# Bank Account Management System in C++

This is a simple console-based Bank Account Management System developed in C++.

## Features

- Create new bank account
- Login with username and password
- Deposit money
- Withdraw money
- Check account balance
- View account details
- View all registered accounts
- CNIC and phone number validation

## Technologies Used

- C++
- Object-Oriented Programming
- Functions
- Arrays
- Input Validation
- Console Application

## How to Run

1. Download or clone this repository.
2. Open the `main.cpp` file in any C++ IDE.
3. Compile and run the program.

## Project Purpose

This project was created to practice basic C++ programming concepts, Object-Oriented Programming, validation, and menu-driven application development.
